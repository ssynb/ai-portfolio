# 苏帅宇 AI 应用开发作品集

这是一个可直接发布到 GitHub Pages 的静态作品集网站。

## 本地查看

直接用浏览器打开：

```text
index.html
```

## GitHub Pages 发布

1. 在 GitHub 新建仓库，例如 `ai-portfolio`。
2. 把本文件夹里的所有文件上传到仓库根目录：`index.html`、`README.md`、`assets/`。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 1-2 分钟，GitHub 会生成公网访问链接。

## 发布前建议替换

- `index.html` 中的 GitHub 链接：搜索 `https://github.com/`。
- `index.html` 中的邮箱占位：搜索 `Email：发布前填写你的邮箱`。
- 如需替换简历，把新的 PDF 覆盖到 `assets/su-shuaiyu-ai-resume.pdf`。
